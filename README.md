# 守护 Guardian

一个帮助养成健康习惯的个人 Web App，部署在手机主屏幕上使用。

## 功能模块

- **睡了吗** — 每天 23:30 全屏强制提醒入睡，支持一次 30 分钟延迟，逃跑需写理由才能解锁其他功能
- **喝了吗** — 定时提醒喝水，每日目标 8 杯，支持手动打卡和间隔时间设置
- **站了吗** — 每小时提醒站立活动，每日目标 8 次，支持手动打卡
- **拉了没** — 每日大便打卡，记录连续天数

## 在线访问

guardian-alert.pages.dev

## 安装到手机

iPhone：Safari 打开网址 → 底部分享按钮 → 添加到主屏幕

Android：Chrome 打开网址 → 右上角菜单 → 添加到主屏幕

## 技术栈

纯 HTML / CSS / JavaScript，无后端，数据存储在浏览器本地 localStorage

部署：GitHub + Cloudflare Pages
